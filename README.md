## Cara Compile Dokumen

Pastikan sudah terinstall **TeX Live** atau distribusi LaTeX lain yang setara.

Langkah kompilasi:

```bash
pdflatex main.tex
```

hasil compile an akan keluar seperti:

```
main.pdf
```
